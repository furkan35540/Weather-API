# 🌦️ Weather API App (Python)

A simple Python application that fetches real-time weather information using the OpenWeather API.

## 📸 Preview

```text
Enter city: London

===== WEATHER =====
📍 City: London
🌡 Temperature: 23°C
🤔 Feels Like: 22°C
☁ Weather: Clear Sky
💧 Humidity: 45%
💨 Wind Speed: 4.8 m/s
```

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡 Current temperature
- 🤔 Feels like temperature
- ☁ Weather description
- 💧 Humidity
- 💨 Wind speed
- ❌ Error handling for invalid city names and API issues

---

## 🛠 Technologies Used

- Python 3
- Requests
- OpenWeather API

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/weather-api-app.git
```

Move into the project directory:

```bash
cd weather-api-app
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Get an API Key

1. Create a free account at OpenWeather.
2. Generate an API key.
3. Replace the API key inside `main.py`.

```python
API_KEY = "YOUR_API_KEY"
```

---

## ▶️ Run the Application

```bash
python main.py
```

Example:

```text
Enter city: Warsaw
```
